# 🌌 GlassyAI Dashboard

GlassyAI is a professional, fully customizable browser new-tab extension designed to supercharge your productivity. It replaces your default new tab with a stunning glassmorphic interface packed with native AI integrations, productivity tools, and rich media backgrounds.

## ✨ Features

- **🤖 Native AI Engine (Jarvis Mode):** Chat with ChatGPT, Gemini, or Claude directly from your search bar. It features Contextual Memory so the AI remembers your conversation history!
- **🎙️ Voice Search:** Integrated Web Speech API allows you to click the microphone icon and speak your prompts naturally.
- **🖼️ Local Media Backgrounds:** Upload massive 4K looping `.mp4` videos or `.jpg` images. Stored entirely locally via IndexedDB for zero latency.
- **⏱️ Productivity Suite:** Features a built-in Pomodoro Focus Timer, ambient Zen Soundscapes (Rain, Fire, Coffee), and a persistent Quick Notes scratchpad.
- **🌦️ Dynamic Weather:** Fetches live weather data based on your location and automatically overlays rain or snow animations on your background depending on the forecast.
- **🗂️ Browser Integration:** Securely accesses your browser logs to display recently closed tabs and your most frequent sites.
- **🛠️ Developer Tools:** Built-in JSON formatter and Base64 encoder/decoder right in your sidebar.

## 🚀 Installation (Developer Mode)

1. Clone this repository or download the ZIP file.
2. Open Chrome (or any Chromium browser) and navigate to `chrome://extensions/`.
3. Enable **Developer mode** in the top right corner.
4. Click **Load unpacked** and select the folder containing the extension files.
5. Open a new tab and enjoy GlassyAI!

## ⚙️ Technologies Used

- **HTML5 & CSS3** (Vanilla, Advanced Glassmorphism)
- **Vanilla JavaScript (ES6+)**
- **Chrome Extensions API** (Tabs, Sessions, History, Storage)
- **Web Speech API** for voice recognition
- **IndexedDB** for local high-fidelity media storage
- **Open-Meteo API** for live weather data
